# Product

## Register

product

## Users

Windows users who run Codex and Claude Code locally and want Feishu/Lark notices when an approval request appears or a task finishes. They may not be comfortable with terminals, tunnels, local services, or manual rule editing.

## Product Purpose

Agent Feishu is a small resident Windows utility. Success means a user can double-click the EXE, scan the Feishu QR flow for a self-built app, add project folders, and then receive Codex/Claude approval and completion notices without learning command-line setup.

## Brand Personality

Calm, practical, trustworthy. The UI should feel like a settings panel for a dependable local tool, not a developer demo or a marketing page.

## Anti-references

Avoid raw command-parameter panels, oversized text areas as the main workflow, web dashboards, localhost setup screens, and fields that look required but are advanced or optional.

## Design Principles

- Lead with the user's job: connect Feishu, add projects, verify with one test.
- Make Codex and Claude support automatic by default.
- Keep optional or advanced behavior visually secondary.
- Use familiar Windows controls and predictable labels.
- Every status message should say what happened and what to do next.

## Accessibility & Inclusion

Use readable system fonts, clear labels, keyboard-focusable controls, restrained color, and no motion-dependent interactions.
